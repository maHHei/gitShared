

1.python修改导入模块的搜索路径（找不到模块的时候就可以修改）
import sys	//导入sys模块
sys.path	//列表，这是Python执行时搜索的文件路径的保存，直接把模块搜索路径加入这里就可以
sys.path.append("")	//添加路径

2.reload重新导入模块（.py文件就是一个模块）
当模块发生改变的时候，得重新导入才会生效

3.判断对象
== ：判断两对象之间内容是否相同
is ：判断两对象是不是同一个对象（id(对象)可以得到对象的值）

4.深拷贝和浅拷贝
a=b //浅拷贝，内存地址一样，即id一样
import copy
c=copy.deepcopy(a)	//深拷贝，复制a内存地址的一份到c内存
